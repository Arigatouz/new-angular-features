Signals All the Way

No more Zone.js hunting for changes. Fine-grained reactivity with signals, computed values, and effects. The framework finally feels reactive.

httpResource

Async data that behaves like a signal. Load, reload, and derive state from HTTP calls without the boilerplate of old patterns.

Native Animations

No more animation modules. Built-in enter and leave hooks let you drop in CSS animations directly. It just works.

About This Build Not Stable

We are pulling Angular straight from the nightly builds on GitHub. That means bleeding edge features, occasional rough edges, and the chance to see where the framework is heading. If something breaks, it is because we are living on the edge. Check out the Examples page to see what we are currently playing with.
