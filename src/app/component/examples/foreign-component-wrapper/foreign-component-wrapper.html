Foreign Component

v22 nightly

Angular v22 introduces foreignImports on @Component — a new mechanism to declare and render components from other frameworks (React, Vue, Web Components) directly in Angular templates. The compiler infrastructure landed in sha-06b004e; full template rendering comes in a future nightly. Below each live demo shows what the adapter pattern will look like.

Experimental — compiler infrastructure only

foreignImports on "@Component" was added as compiler infrastructure in PR #68643 (May 18). The live components above use afterNextRender +ß createRoot / createApp directly — the adapter pattern that foreignImports will automate once rendering lands.
